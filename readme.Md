# Personal Portfolio Website

## Overview

This is a personal portfolio website developed by **Sana Sajid** using HTML and CSS. The website serves as an online portfolio to showcase personal information, educational background, technical skills, and projects.

The portfolio is designed with a clean and responsive layout to provide a user-friendly experience across different devices.

---

## Features

* Responsive Navigation Bar
* Personal Introduction Section
* Education Page
* Skills Page
* Projects Page
* Social Media Links
* Email Contact Button
* Mobile-Friendly Design

---

## Pages Included

### Home

Introduces Sana Sajid, including a profile image, professional summary, contact options, and social media links.

### Education

Displays academic background, qualifications, and educational achievements.

### Skills

Highlights technical and soft skills related to web development and computer science.

### Projects

Showcases projects completed during the learning journey and portfolio development process.
